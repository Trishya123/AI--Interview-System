# AI-Interview-System
This project is an AI-powered smart interview system that asks different questions based on the job you are applying for. Instead of asking generic questions, the system intelligently generates interview questions based on a candidate's resume, the job role they are applying for, and a specialized knowledge base created from machine learning textbooks using RAG (Retrieval-Augmented Generation) — a technique where the AI searches through a knowledge base of ML textbooks to find relevant content before generating questions, ensuring questions are accurate and grounded in real knowledge rather than made up. This allows the interview process to feel more natural, relevant, and tailored to each candidate's background and expertise, while helping recruiters assess technical skills more effectively.
